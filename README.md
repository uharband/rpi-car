# rpi-car

raspberry-pi car

install

```
cd ~
git clone https://github.com/uharband/rpi-car.git
cd rpi-car/
chmod +x install.sh server/install.sh server/lib/audio/startAudioStreaming.sh server/lib/audio/stopAudioStreaming.sh video/install.sh audio/install.sh access_point/install.sh access_point/rPi3-ap-setup.sh 
./install
sudo reboot
```
